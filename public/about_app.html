*ต้องเข้าสู่ระบบถึงจะใช้เมนูนี้ได้

การใช้แชทบอท

เมื่อทำการเข้าใช้แชทบอทจะขึ้นหน้าจอดังรูป

ถ้าต้องการแชทสนทนากับแชทบอทให้กดตรงที่ทำกรอบสีแดงไว้

เมื่อกดแล้วก็จะสามารถสนทนากับแชทบอทได้ตามปกติดังภาพ

บันทึกการดื่มน้ำและปัสสาวะ*

เมื่อทำการเข้าสู่หน้าบันทึกการดื่มน้ำและปัสสาวะจะขึ้นหน้าจอ ดังรูป

1. เลือกวันที่จะทำการบันทึกโดย กดที่วันที่ในกรอบสีแดง และกดค้นหาในกรอบสีชมพู

2. ให้กดเครื่องหมาย + ในกรอบสีเขียวเพื่อเริ่มการบันทึก

เมื่อทำการกดแล้วจะเข้าสู่หน้าการบันทึก ดังรูป

1. การเลือกวันที่จะทำการบันทึกใหม่ โดยกดที่วันที่ในกรอบสีแดง แต่ถ้าไม่ต้องการก็สามารถใช้วันที่แสดงได้เลย

2. การเลือกเวลาที่จะทำการบันทึกใหม่ โดยกดที่เวลาในกรอบสีชมพู แต่ถ้าไม่ต้องการก็สามารถใช้เวลาที่แสดงได้เลย

3. การเปลี่ยนน้ำหนักที่จะทำการบันทึกใหม่ โดยกดที่น้ำหนักในกรอบสีเขียว แต่ถ้าไม่ต้องการก็สามารถใช้น้ำหนักที่แสดงได้เลย

4. เลื่อกประเภทการบันทึกโดยกดไปที่ปุ่มวงกลมในกรอบสีน้ำเงิน สามารถกดได้หลายประเภทในทีเดียว

เมื่อทำการกดประเภทดื่มน้ำจะแสดงผล ดังรูป

1. ใส่ชนิดของน้ำที่ดื่มในกรอบสีแดง

2. ใส่ปริมาณของน้ำที่ดื่มในกรอบสีชมพู

3. ถ้ามีชนิดของน้ำมากกว่า 1 ปรเภทสามารถกดเพิ่มในกรอบสีน้ำเงิน

4. ถ้าต้องการลบบันทึกนั้นให้กด - ในกรอบสีเขียว

เมื่อทำการกดประเภทสวนปัสสาวะ ปัสสาวะด้วนตนเอง และอื่นๆ จะแสดงผล ดังรูป

ประเภทสวนปัสสาวะ : ใส่ปริมาณของปัสสาวะที่ได้ในกรอบสีแดง

ประเภทปัสสาวะด้วยตนเอง : ใส่ปริมาณของปัสสาวะที่ได้ในกรอบสีชมพู

ประเภทอื่นๆ : ใส่ข้อความหรือหมายเหตุอื่นๆได้ในกรอบสีเขียว

เมื่อทำการบันทึกเสร็จสิ้นให้กด บันทึก ในกรอบสีน้ำเงิน

ถ้าต้องการยกเลิกการบันทึกนี้ให้กด ยกเลิก ในกรอบสีเหลือง

เมื่อทำการกดยืนยัน หรือ ยกเลิกจะทำการกลับสู่หน้าบันทึกการดื่มน้ำและปัสสาวะ ดังรูป

กรอบสีดำ : แสดงปริมาณน้ำที่เข้าสู่ร่างกายทั้งหมด

กรอบสีเทา : แสดงปริมาณน้ำที่ออกจากร่างกายทั้งหมด

กรอบสีแดง : แสดงบันทึกของการสวนปัสสาวะ

กรอบสีชมพู : แสดงบันทึกของการปัสสาวะด้วยตนเอง

กรอบสีเขียว : แสดงบันทึกหมายเหตุอื่นๆ

กรอบสีน้ำเงิน : แสดงบันทึกของน้ำที่ดื่ม

กรอบสีฟ้า : แสดงเวลาบันทึกประเภทนั้นๆ โดยถ้าบันทึกในเวลาเดียวกันจะอยู่ด้วยกัน แต่ถ้าคนละเวลาจะแยกออกจากกัน

ถ้าต้องการลบบันทึกทั้งหมดของช่างเวลานั้นๆให้กด ลบ ในกรอบสีส้ม

ถ้าต้องการแก้ไขบันทึกทั้งหมดของช่างเวลานั้นๆให้กด แก้ใข ในกรอบสีม่วง โดยจะทำการย้อนกลับไปที่หน้าการบันทึก

คู่มือการสวนปัสสาวะ

เมื่อทำการเข้าสู่หน้าคู่มือการสวนปัสสาวะจะขึ้นหน้าจอ ดังรูป

มีให้เลือกกด 2 ส่วนคือ

1. อุปกรณ์สวนปัสสาวะ โดยให้ทำการกดในกรอบสีแดง

2. ขั้นตอนสวนปัสสาวะ โดยให้ทำการกดในกรอบสีชมพู

เมื่อทำการกด อุปกรณ์สวนปัสสาวะ จะแสดงหน้าจอดังรูป

ในหน้านี้จะแสดงรายละเอียดของอุปกรณทั้งหมดที่ใช้ในการสวนปัสสาวะ

เมื่อทำการกด ขั้นตอนสวนปัสสาวะ จะแสดงหน้าจอดังรูป

กดตรง เพศหญิง ในกรอบสีแดง เพื่อแสดงวิธีสวนปัสสาวะของเพศหญิง

กดตรง เพศชาย ในกรอบสีน้ำเงิน เพื่อแสดงวิธีสวนปัสสาวะของเพศชาย

*ถ้าเข้าสู่ระบบแล้วจะแสดงวิธีสวนปัสสาวะในเพศของผู้ใช้โดยอัตโนมัติ

ประวัติส่วนตัว

เมื่อทำการเข้าสู่หน้าประวัติส่วนตัวจะขึ้นหน้าจอ ดังรูป

ในหน้านี้จะแสดงประวัติส่วนตัวของผู้ใช้โดยไม่สามารถทำการแก้ไขได้

ยกเว้นในส่วนของส่วนสูงและน้ำหนัก ที่สามารถแก้ไขได้ โดยการกรอกในกรอบสีแดง

เมื่อทำการแก้ไขแล้วให้กด บันทึกข้อมูล ในกรอบสีชมพู
